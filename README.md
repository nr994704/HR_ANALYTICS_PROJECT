# HR_ANALYTICS_PROJECT

🔹 Project Overview

This project focuses on employee analytics, helping organizations understand workforce trends, employee attrition, job satisfaction, performance, and other key HR metrics through an interactive Power BI dashboard.

🛠️ Tools & Technologies

📑 Excel – Data Cleaning & Preparation
🗄️ SQL – Data Extraction & Analysis
📊 Power BI – Interactive Dashboard & Data Visualization

📈 Key Insights

Employee Attrition Analysis
Department-wise Employee Distribution
Gender & Age Analysis
Job Role & Education Analysis
Job Satisfaction Analysis
Salary & Experience Insights
Interactive Filters (Department, Gender, Education, etc.)
KPI Cards (Total Employees, Attrition Rate, Average Age, Average Salary)
